# investment-research-reports

Investment research reports — daily market analysis and monthly portfolio recommendations.

## Structure

```
reports/
└── YYYY/
    └── MM/
        ├── portfolio.md              # Monthly portfolio recommendation
        ├── portfolio_allocation.png  # Portfolio allocation chart
        ├── momentum_scores.png       # Universe momentum chart
        └── DD/
            ├── daily_report.md       # Daily market report
            ├── market_overview.png   # 5-day normalized price chart
            └── sector_performance.png  # Sector daily returns chart
```

## Naming Conventions

| Report | Path | Frequency |
|--------|------|-----------|
| Daily market report | `reports/YYYY/MM/DD/daily_report.md` | Daily (weekdays) |
| Daily price chart | `reports/YYYY/MM/DD/market_overview.png` | Daily (weekdays) |
| Daily sector chart | `reports/YYYY/MM/DD/sector_performance.png` | Daily (weekdays) |
| Monthly portfolio | `reports/YYYY/MM/portfolio.md` | Monthly (end of month) |
| Portfolio allocation | `reports/YYYY/MM/portfolio_allocation.png` | Monthly |
| Momentum chart | `reports/YYYY/MM/momentum_scores.png` | Monthly |

## Report Contents

### Daily Report (`daily_report.md`)

- Market summary table: key ETFs and assets with 5-day returns
- Normalized price chart (5-day)
- Sector performance bar chart (daily returns)
- Sector detail table

### Monthly Portfolio (`portfolio.md`)

- Top N recommended holdings (equal-weight)
- Risk-adjusted momentum scoring methodology
- Full universe ranking table
- Allocation pie chart
- Momentum bar chart

## Generation

Reports are generated by the [investment-research-code](https://github.com/blackthorncap/investment-research-code) repository using:

```bash
# Daily report
uv run daily-report --reports-dir path/to/this/repo

# Monthly portfolio
uv run monthly-portfolio --reports-dir path/to/this/repo
```
